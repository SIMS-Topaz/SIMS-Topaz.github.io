Topaz

Version 1.0
Find out the coolest things nearby
Topaz is a free, android application for helping you find cool things nearby. Searches by places, friends or tags can be performed!
  • Easy to express yourself about different places
  • Search messages by places
  • Search messages by friends
  • Search messages by tags
Requirements:
Requires Android 2.3 and higher. WiFi, Edge, or 3G network connection required.

About

Topaz is a fully native android application perfect for sharing your thoughts and finding out news. Communication and real time news sharing is the key of the application.
